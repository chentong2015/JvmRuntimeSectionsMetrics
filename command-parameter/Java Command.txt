TODO. Java Command命令全集: 支持运行时参数配置
https://docs.oracle.com/en/java/javase/17/docs/specs/man/java.html

> javac HelloWorld.java    编辑源代码，生成HelloWorld.class中间代码
> java HelloWorld          运行程序生成的.class文件
> java -jar HelloWorld.jar 运行项目打包的jar包
> java -jar project.war    运行项目打包的war包

> java -cp my-project.jar com.my_package.MyClass 运行jar包中指定启动类
> java -cp my-project.jar;dependency-jars/* com.example.MyClass
        -cp -classpath路径加载自定义类型(编译好的.class文件)
        ; separate list of directories 提供多个classpath加载路径
> java -cp ../target/classes -Dfile.encoding=UTF-8 com.example.MainClass
> java -cp target/classes -XX:-RestrictReservedStack --enable-preview $1
        $1 参数传递要执行的main类

> nohup java -jar application.jar > app.log 2>&1 &
  > app.log 把标准输出写到 app.log
  2>&1 把标准错误也重定向到标准输出(app.log)
  & 把进程放到后台运行, 避免因为终端关闭而终止
